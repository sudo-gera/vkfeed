# vkfeed
## instruction for installing on android:
###### open playmarket
###### install termux
###### install termux-api
###### check if your device is connected to internet
###### open termux and type theese comands:
```
pkg install git
git clone https://github.com/sudo-gera/vkfeed
vkfeed/install
```
# run vkfeed:
###### open termux and type:
```
vkfeed
```
